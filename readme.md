# 🌊 Phish Tank: Junior Defender

**Phish Tank** is a simple browser based educational game designed to teach children (and beginners) how to identify phishing threats in their email inboxes. By simulating a "digital fish tank," players learn to spot red flags in sender addresses and subject lines while maintaining a secure system.

## 🚀 Purpose
Email phishing is the #1 way cybercriminals gain access to personal information. This game aims to:
- Build a "security-first" mindset.
- Teach players to verify sender domains (e.g., `google.com` vs. `g0ogle.net`).
- Familiarize users with common scam tactics like fake prizes and urgent alerts.

## 🎮 How to Play
1.  **Start:** Enter your name to begin.
2.  **Training:** Review the "Domain Training" guide to see the difference between healthy and malicious emails.
3.  **Inspect:** Emails float through the tank. **Hover your mouse** over an email to pause it for closer inspection.
4.  **Action:**
    *   Click **SAFE** for legitimate emails to deliver them to the secure computer mailbox.
    *   Click **SCAM** to report phishing attempts.
5.  **Win Condition:** Successfully identify **5 emails** to secure the tank!
6.  **Loss Condition:** If **5 phishing emails** slip through or safe mail is misidentified, the player loses.

## 🛠️ Built With
- **HTML5:** Structure and content.
- **CSS3:** Animations (swaying plants, rising bubbles) and fish tank aesthetics.
- **JavaScript (Vanilla):** Game logic, collision detection (lane management), and state tracking.

## 📁 Installation & Setup
No server or installation is required!
1. Download the `phish-tank.html` file.
2. Double-click the file to open it in any modern web browser (Chrome, Firefox, Edge, or Safari).

## 🧠 Educational Red Flags Included
Players must watch out for:
- **Character Swapping:** `amaz0n` instead of `amazon`.
- **Suspicious Top-Level Domains:** `.biz`, `.net`, and `.xyz`.
- **Urgency Hooks:** "Action Required," "Hold," and "Prize."
